# Screen Time Analysis 📊

## 📌 Objective
Analyze the impact of screen time on productivity, sleep, and cognitive performance.

## 🛠 Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook

## 📂 Project Structure
- data → raw and cleaned datasets
- notebook → data analysis notebook
- powerbi → dashboard file
- presentation → project PPT

## 🔍 Key Insights
- High screen time reduces productivity
- Social media negatively impacts performance
- Sleep improves cognitive score
- Study hours increase productivity

## 📊 Components
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Power BI Dashboard
- Presentation
