# 📊 Screen Time Analysis & Cognitive Efficiency

## 📌 Project Overview

This project focuses on analyzing how screen time and digital behavior impact **productivity, sleep patterns, and cognitive performance**.

With the increasing use of smartphones and social media, understanding these relationships is essential for improving **focus, efficiency, and overall well-being**.

The project follows a complete **end-to-end data analytics pipeline**, including:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Dashboard Creation
* Insight Presentation

---

## 🎯 Objectives

* Analyze the relationship between **screen time and productivity**
* Study the impact of **social media usage on cognitive performance**
* Understand how **sleep influences efficiency**
* Identify key factors affecting **student performance**
* Provide actionable insights for better time management
